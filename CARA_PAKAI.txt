📋 CARA CEPAT MENJALANKAN APLIKASI:

1. BUKA TERMINAL:
   Ctrl+Alt+T (Linux) atau buka Command Prompt/Terminal

2. MASUK KE FOLDER APLIKASI:
   cd /home/ikanx101/Documents/Aplikasi\ 1

3. JALANKAN SCRIPT:
   ./start.sh

   ATAU secara manual:
   npm install   (jika pertama kali)
   npm start     (untuk menjalankan)

4. BUKA BROWSER:
    Buka aplikasi sesuai konfigurasi server

5. UNTUK BERHENTI:
   Tekan Ctrl+C di terminal
